
# Ejercicio: Tienda de Proteínas

## Descripción

El objetivo de este ejercicio es crear una página web responsiva y funcional para una **Tienda de Proteínas** utilizando **Bootstrap**, aplicando los conocimientos adquiridos en el tema.

Deberás implementar las siguientes secciones en la página:

### Requisitos

1. **Encabezado con Barra de Navegación**:
   - Debe incluir:
     - Logotipo (puede ser texto).
     - Un menú con enlaces a las secciones Inicio, Productos y Contacto.
     - El menú debe ser responsivo, mostrando un botón desplegable en dispositivos móviles.

2. **Sección de Productos**:
   - Utiliza un diseño de cuadrícula para mostrar al menos tres productos.
   - Cada producto debe incluir:
     - Una imagen representativa.
     - Nombre del producto.
     - Precio.
     - Un botón para ver más detalles (implementa un modal).

3. **Formulario de Contacto**:
   - Incluye los siguientes campos:
     - Nombre.
     - Correo electrónico.
     - Mensaje.
   - Implementa validación visual para estos campos.

4. **Pie de Página**:
   - Contenido obligatorio:
     - Información de copyright.
     - Enlaces a redes sociales.

5. **Efectos Visuales**:
   - Implementa transiciones en los botones.
   - Añade una animación al modal al abrirse.

### Nota
No habrá archivos base para este ejercicio. Debes crear los archivos `index.html` y `styles.css` desde cero.
