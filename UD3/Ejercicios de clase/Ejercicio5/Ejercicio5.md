# Ejercicio Práctico: Biblioteca Responsive con Bootstrap

En este ejercicio, desarrollarás una página web para una biblioteca. Debes aplicar las clases utilitarias de Bootstrap para crear un diseño completamente responsive, utilizando el sistema de cuadrícula, espaciado, alineación, visibilidad, y control de tamaño de texto e imágenes.

## Requisitos

1. Crea un encabezado que contenga el título de la biblioteca y un subtítulo. Aplica márgenes y padding adecuados.
2. Dentro de la sección principal:
   - Diseña una cuadrícula con información sobre las secciones de la biblioteca (por ejemplo: "Ficción", "No Ficción", "Revistas", etc.).
   - Asegúrate de que las tarjetas de las secciones se adapten correctamente a diferentes tamaños de pantalla.
3. Implementa un sistema de navegación con enlaces a las distintas secciones de la página.
4. Añade un contenedor con una lista de "Novedades" que se muestre como una lista vertical en dispositivos pequeños y como una cuadrícula en dispositivos medianos y grandes.
5. Usa imágenes con tamaño responsivo para ilustrar las secciones de la biblioteca y las novedades.
6. Añade un pie de página con enlaces a las redes sociales y un mensaje que se oculte en dispositivos pequeños.
7. Aplica control de texto para destacar el nombre de la biblioteca y los títulos de las secciones.

## Restricciones

- No puedes utilizar estilos CSS personalizados, solo clases de Bootstrap.
- Asegúrate de que la página sea completamente responsive y que los elementos se ajusten adecuadamente en todos los tamaños de pantalla.

## Entregable

Entrega el archivo HTML completo que cumpla con todos los requisitos.
