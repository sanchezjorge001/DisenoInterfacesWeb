
# Ejercicio Completo: Página de Eventos Culturales

## Descripción

El objetivo de este ejercicio es crear una página web responsiva y funcional utilizando **Bootstrap**, donde se pongan en práctica todos los conceptos del tema, incluyendo:
- **Sistema de cuadrícula** para el diseño responsivo.
- **Componentes** como barra de navegación, tarjetas, formulario, modales y pie de página.
- **Animaciones y transiciones** aplicadas a los elementos interactivos.
- **Clases utilitarias** para ajustar espaciado, visibilidad y alineación.

La temática del ejercicio es una página de **eventos culturales**, que debe incluir las siguientes secciones:

### Requisitos

1. **Encabezado con Barra de Navegación**:
   - Debe incluir:
     - Logotipo (puede ser texto).
     - Menú con al menos tres enlaces (Inicio, Eventos, Contacto).
     - Un menú desplegable que debe funcionar en dispositivos móviles.
   - Implementa una animación para el menú desplegable.

2. **Sección de Eventos**:
   - Utiliza un diseño de **cuadrícula** para mostrar al menos tres tarjetas.
   - Cada tarjeta debe contener:
     - Una imagen representativa del evento.
     - Nombre, fecha y ubicación del evento.
     - Un botón para mostrar más detalles (que abre un modal, implementado por el alumno).

3. **Formulario de Contacto**:
   - Incluye los siguientes campos:
     - Nombre.
     - Correo electrónico.
     - Mensaje.
   - Añade validación visual con **clases utilitarias**.

4. **Pie de Página**:
   - Contenido obligatorio:
     - Información de copyright.
     - Enlaces a redes sociales.
   - Debe ser responsivo y alinearse al centro.

5. **Efectos Visuales**:
   - Añade una animación cuando se muestra el modal.
   - Implementa transiciones suaves en los botones y enlaces.
