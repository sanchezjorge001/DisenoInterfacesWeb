
# Ejercicio Práctico: Integración de Animaciones y Transiciones en Bootstrap

Este ejercicio tiene como objetivo practicar las animaciones y transiciones en Bootstrap, incluyendo la personalización mediante CSS y la integración con bibliotecas externas. 

## Enunciado
Crea una página web utilizando Bootstrap con los siguientes requisitos:

### 1. Modal con Animación Personalizada
- Implementa un modal que utilice las clases de Bootstrap `fade` y `modal`.
- Personaliza la animación del modal para que tenga un efecto de deslizamiento hacia abajo al abrirse y hacia arriba al cerrarse.
- Ajusta la duración de la transición a 0.8 segundos.

### 2. Dropdown con Transición Personalizada
- Crea un menú desplegable (dropdown) utilizando Bootstrap.
- Personaliza el efecto del dropdown para que se deslice suavemente hacia abajo al abrirse.
- Utiliza transiciones CSS con una duración de 0.5 segundos.

### 3. Sección Expandible con `collapse`
- Añade una sección en la página que utilice la clase `collapse` de Bootstrap.
- Personaliza la animación de la sección para que se expanda o contraiga suavemente en 0.6 segundos.

### 4. Integración con Animate.css
- Utiliza Animate.css para añadir un efecto de rebote (`bounce`) a un botón al hacer clic.
- Descarga Animate.css o incluye el enlace CDN en tu proyecto.

### 5. Elementos Interactivos con Clases Utilitarias
- Añade un elemento que utilice las clases `d-none` y `d-block` para controlar su visibilidad.
- Implementa un botón que cambie el estado del elemento entre visible y oculto con una transición suave.

## Requisitos Técnicos
1. Utiliza Bootstrap 5 y personaliza las animaciones utilizando CSS.
2. Añade un diseño responsivo para que la página se vea bien en dispositivos móviles.
3. Usa un archivo CSS externo para la personalización de las animaciones.
4. Incluye un archivo HTML funcional con todos los elementos requeridos.
5. Documenta brevemente el funcionamiento de cada parte del código.

## Entregables
- Un archivo `index.html` que cumpla con todos los requisitos.
- Un archivo `styles.css` con las personalizaciones necesarias.
- Enlace funcional a Animate.css si se usa desde CDN.

---

### Evaluación
1. **Modal con Animación Personalizada** (25%)
2. **Dropdown con Transición Personalizada** (25%)
3. **Sección Expandible con `collapse`** (20%)
4. **Integración con Animate.css** (20%)
5. **Uso de Clases Utilitarias** (10%)

Sube tu proyecto comprimido en un archivo `.zip` con los nombres `index.html` y `styles.css`. Asegúrate de que todo funcione correctamente antes de la entrega.
