# Ejercicio Práctico: Diseño Responsive con Bootstrap

En este ejercicio, practicarás las clases utilitarias de Bootstrap para espaciado, alineación, visibilidad y control de tamaño de texto e imágenes. Crearás una página web responsive que se adapte automáticamente a diferentes tamaños de pantalla.

## Requisitos

1. **Sistema de Espaciado**:
   - Crea un encabezado (`<header>`) con una clase que aplique márgenes y padding específicos.
   - Añade un párrafo dentro del encabezado con `mt-4` y `mb-2`, además de padding horizontal `px-3`.

2. **Alineación de Texto**:
   - Dentro del cuerpo principal (`<main>`), añade tres secciones con los textos alineados a la izquierda, al centro y a la derecha usando las clases `text-start`, `text-center`, y `text-end`.

3. **Alineación Flexbox**:
   - Añade un contenedor con tres cajas dentro, alineadas con `justify-content-between`.

4. **Clases de Visibilidad**:
   - Incluye un mensaje que solo sea visible en pantallas medianas o más grandes usando `d-none d-md-block`.
   - Añade una sección que se oculte en pantallas grandes con la clase `d-lg-none`.

5. **Control de Tamaño de Texto**:
   - Crea un párrafo con texto muy grande usando `fs-1` y otro con texto pequeño usando `fs-6`.

6. **Control de Imágenes**:
   - Inserta una imagen utilizando la clase `img-fluid` para que se ajuste automáticamente al tamaño del contenedor.

## Restricciones
- Usa solo clases de Bootstrap para aplicar los estilos. No está permitido usar estilos CSS personalizados en este ejercicio.
